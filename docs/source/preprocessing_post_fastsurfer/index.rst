======================
preprocessing_post_fastsurfer
======================

Overview
--------

This package contains functions for manipulating MRI data after it has been processed using fastsurfer. This includes alignment, cropping, extraction and mesh sampling. See preprocess_hcampus_1.5_cohort.ipynb for an example of how the pipeline was used in the project.
